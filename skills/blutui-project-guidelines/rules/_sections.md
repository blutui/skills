# Sections

This file defines all sections, their ordering, impact levels, and descriptions.
The section ID (in parentheses) is the filename prefix used to group rules.

---

## 1. Foundation

**Impact:** CRITICAL  
**Description:** The base foundation that must be incorparated in a Blutui project.

## 2. Components

**Impact:** CRITICAL  
**Description:** Standard rules and instructions on how to implement frontend components with Blutui compatible syntax.

## 3. Model Context Protocol (MCP)

**Impact:** MEDIUM  
**Description:** Blutui MCP standards and available tools guide.

## 4. Canvas

**Impact:** CRITICAL
**Description:** Blutui template engine guide.

## 5. Courier

**Impact:** MEDIUM  
**Description:** Blutui command line interface guide.

## 6. Cassettes

**Impact:** LOW 
**Description:** Blutui project-specific version control guide.

## 7. Collections

**Impact:** CRITICAL  
**Description:** A guide to add anf manage data in a blutui project.
